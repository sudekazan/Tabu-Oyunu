# Tabu Oyunu

Eğlenceli ve bağımlılık yaratan kelime tahmin oyunu! Arkadaşlarınızla oynayın, puan toplayın ve eğlenin.

## Özellikler

- 🎯 **Çoklu Kategori**: Film, dizi, marka, çocuk, hayvan gibi çeşitli kategoriler
- 👥 **Çoklu Oyuncu**: 2-8 oyuncu desteği
- ⏱️ **Zamanlı Oyun**: Her tur için ayarlanabilir süre
- 💎 **Premium Özellikler**: Reklamsız deneyim, sınırsız pas hakkı
- 🎨 **Modern Tasarım**: Şık ve kullanıcı dostu arayüz
- 🌍 **Türkçe Destek**: Tamamen Türkçe arayüz

## Teknik Özellikler

- **Platform**: iOS & Android
- **Framework**: Flutter
- **Dil**: Dart
- **Reklam**: Google AdMob
- **Satın Alma**: In-App Purchase
- **Veri Saklama**: Shared Preferences

## Kurulum

1. Flutter SDK'nın kurulu olduğundan emin olun
2. Projeyi klonlayın
3. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```
4. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```
